# Sharacle

Public feedback portal for [sharacle.com](https://sharacle.com).

This repository exists to:

- share a high-level overview of the product,
- collect user and developer feedback in one place,
- provide a transparent public issue tracker.

## About Sharacle

Sharacle is an anxiety-free social platform focused on sharing and cherishing
meaningful moments.

This is **not** the application source code repository. The production codebase
is private.

## What To Use This Repo For

- report bugs you notice while using Sharacle,
- suggest product or UX improvements,
- share developer-oriented feedback (for example, integration or platform
  behavior notes).

## How To Submit Feedback

- Open a GitHub issue using one of the templates in this repository.
- For private or sensitive reports, use in-app reporting or email contact
  instead of posting public details.

## Response Expectations

- We review issues regularly.
- Initial issue triage target: within 7 days.
- We may ask follow-up questions before confirming changes.
- Not every request will be accepted, but all useful feedback is appreciated.

## Security And Sensitive Reports

Please do not post security vulnerabilities or personal/sensitive account
information in public issues. Use private contact channels for those reports.
