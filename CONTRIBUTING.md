# Contributing

Thanks for helping improve Sharacle.

This repository is a public feedback portal, not an open-source codebase. We
welcome issues and suggestions, but we do not accept code pull requests for the
private application currently.

## Before Opening An Issue

- Check existing issues to avoid duplicates.
- Keep reports specific and reproducible when possible.
- Remove personal or sensitive information.

## Good Issue Content

- clear summary,
- expected behavior,
- actual behavior,
- steps to reproduce,
- screenshots or short recordings (if relevant),
- page or URL where it happened,
- device, browser, and OS details,
- whether you were using Sharacle in-browser or as an installed PWA.

Because Sharacle is a website/PWA, browser and page context are usually more
useful than a separate app version number.

## What Happens Next

- Maintainers triage and label issues.
- We may request additional context.
- Accepted items are moved into internal planning and implementation.
