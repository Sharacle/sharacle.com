# Support

Use the right channel for faster help:

| Need                                    | Best Channel                                                |
| --------------------------------------- | ----------------------------------------------------------- |
| Public bug report or product suggestion | GitHub Issues                                               |
| General usage question                  | GitHub Issues (`General Question` template)                 |
| Account-specific help                   | In-app support/report flow                                  |
| Sensitive or private concern            | Email: <sharacle.services@gmail.com>                        |
| Security vulnerability                  | Email: <sharacle.services@gmail.com> (do not post publicly) |

For public bug reports, include the page or URL, your device, OS, browser, and
whether you were using Sharacle in-browser or as an installed PWA.
